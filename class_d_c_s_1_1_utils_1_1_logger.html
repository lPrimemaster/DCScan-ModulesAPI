enum DCS::Utils::Logger::Verbosity
strong

Contains the possible verbosity levels for the Logger.

Passing a value to Logger::Init will prevent lower priority messages from being displayed in the console. The file output (Logger::handle) is not affected by this, printing every event.

Enumerator
CRITICAL 

Enables console output only for critical messages.

ERROR 

Enables console output for error and higher priority messages.

WARNING 

Enables console output for warning and higher priority messages.

MESSAGE 

Enables console output for status and higher priority messages.

DEBUG 

Enables console output for debug and higher priority messages.
