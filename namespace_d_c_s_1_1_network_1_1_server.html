enum DCS::Network::Server::StopMode
strong

Dictates how the server thread should stop.

Enumerator
IMMEDIATE 

Stop server thread without waiting for the client to disconnect (force shutdown server-side).

WAIT 

Stop server after waiting for the client to disconnect.

void DCS::Network::Server::StartThread ( Socket  client,
OnDataReceivedCallback  drc 
)

Starts the thread for the server and waits for client data.

The client is responsible for making the first request. Data can be sent via the OnDataReceivedCallback callback function (see SendData(Socket, const unsigned char*, DCS::i32))

Parameters
clientThe client connection Socket.
drcThe function to be called when a full message is received. A lambda expression can be used as the callback, provided it does not capture.

DCS::Network::Socket DCS::Network::Server::WaitForConnection ( Socket  server)

Waits for a client connection on a valid server Socket, blocking code execution on the running thread.

The server Socket is destroyed after the client connects. Only one client connection is allowed at a given time.

Parameters
serverThe server Socket.
Returns
Socket containing the client connection.
