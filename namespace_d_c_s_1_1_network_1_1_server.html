void DCS::Network::Server::StartThread ( Socket  client,
OnDataReceivedCallback  drc 
)

Starts the thread for the server and waits for client data.

The client is responsible for making the first request. Data can be sent via the OnDataReceivedCallback callback function (see SendData(Socket, const char*, DCS::i64))

Parameters
clientThe client connection Socket.
drcThe function to be called when a full message is received. Expects text data. A lambda expression can be used as the callback, provided it does not capture.
Examples:
sockets/echo_server.cpp.

DCS::Network::Socket DCS::Network::Server::WaitForConnection ( Socket  server)

Waits for a client connection on a valid server Socket, blocking code execution on the running thread.

The server Socket is destroyed after the client connects. Only one client connection is allowed at a given time.

Parameters
