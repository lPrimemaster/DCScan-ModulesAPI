Detailed Description

Handles default messaging via a simple protocol.

Messages sent via network can be of any type (for example JSON), however, the Messaging namespace provides a default simple messaging style used by the GUI when contacting the server. This implementation attempts to reduce the size of the sent messages, maximizing network functionality.
